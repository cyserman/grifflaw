# Backup Directory

This directory contains a backup of the original Griffiths Law Office website, preserved before the site replacement.

## Contents

- **old-site/** - Complete backup of the original Next.js law firm website
  - All pages (home, attorney, war-room, contact)
  - All components (Navigation, FloatingActionButton, SchedulerModal, ReviewTicker)
  - Configuration files
  - Documentation

## Backup Date
Created: January 14, 2026

## Original Site Features
The backed up site was a fully functional Next.js 15 law firm website with:
- Dark mode design with gold accents
- Smart scheduler integration (Cal.com ready)
- Social proof marquee (review ticker)
- The War Room (Pro Se resource center)
- Location & maps page
- Mobile-first responsive design
- SEO optimized

## Restoration
To restore the original site:
1. Copy contents from `backup/old-site/` back to the root directory
2. Run `npm install` to install dependencies
3. Run `npm run dev` to start the development server
